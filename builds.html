DEVELOPMENT BUILDS

I make a new build of the game every time I think theres a substantial enough of an improvement from the last one.

Builds with underlined titles have showcase videos attached. Click them to view it.




SIXTH BUILD

November 6, 2024

Many many, many things were added between the nearly 4 months that had passed since the last build. This build was originally supposed to follow up the last build, where the player would raid the "Wing of Ares", and to do that I needed to finish the fps system. However, as time when on, and the scope of the game only got bigger, I constantly kept improving what I already had. Constantly kept polishing, and experimenting with different gameplay styles. Eventually, I formed a concrete enough of a foundation, and was able to finish the combat system. This build also includes other things like a complete tutorial sequence, artificial intelligence for NPC's and enemies, and many smaller things that have been added throughout this time. I'm still going to wait for the 7th build before making another showcase. You can read the devlog if you are interested in what else I added.

FIFTH BUILD

July 29, 2024

Focused around an in-depth space combat system, lore cutscenes, and heavy refinement of that intro mission. I also gave a lot more depth to the FPS system by adding secondary weapons (grenades and throwing knives), levels and skills, and inventory systems.

FOURTH BUILD

June 12, 2024

Included a showcase with parts of the last build, but mostly focused on a super rough and early intro mission for the game that was planned to serve as the tutorial. A mission that has completely been scraped as of writing this. Also added the first FPS animations, and a better FPS system (still basic).

THIRD BUILD

May 19, 2024

This build added a landing sequence, launching sequence, and an immersive orbiting sequence. Along with a custom VHS shader for the game. I experimented with a HUD rendered entirely in 3D space. I have since reverted back to a typical 2D user interface.

SECOND BUILD

April 28, 2024

Showcasing the earliest form of an actual game, with a minigame. Improved the experimental start screen. There was still not a clear enough direction of a main gameplay loop at this time, but there was a very bare-bones FPS system.

FIRST BUILD

April 21, 2024

Despite development starting a month prior, this was the very first build of the game. Mostly was created to make sure I hadn't screwed up the project settings (like for my last project). It included a rough stylized start screen, and the basics of all games. The game had a pixelation shader applied to everything, that is now being used for other cases.
