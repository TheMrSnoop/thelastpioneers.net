Summer 2024 Developer Update

7.29.2024

GAME DEV

MARCH 2024

V.DEV-3.24

  1. Created the general idea for the game
  2. Made lots of different prototype maps, and features

APRIL 2024

V.DEV-4.24

  1. Created a minigame task and a framework for future ones
  2. Added the framework for FPS gameplay
  3. Made both the 1st, and 2nd build of the game.
  4. Refined the in-game menus

MAY 2024

V.DEV-5.24

  1. Refined the moon base map
  2. Created the first FPS animations
  3. Made a custom VHS styled shader for the game.
  4. Made the player HUD 3D
  5. Refined the in-game menus
  6. Made the 3rd build of the game.

JUNE 2024

V.DEV-6.24

  1. Added a launch sequence intro mission, which is likely to be unused as of writing this.
  2. Made the first FPS animations
  3. Refined the in-game menus
  4. Made the 4th build of the game.

JULY 2024

V.DEV-7.24

  1. Tied weapon attributes such as weapon spread, accuracy, and range to the players current combat level. This includes a leveling system with different sublevels.
  2. Added a relatively in-depth space combat system for the "Interception" core gamemode.
  3. Created a basic inventory and armor mechanic.
  4. Heavily refined the Houston Launchsite Map, that is now likely to be unused as of writing this.
  5. Added a variety of special weapons, like grenades and throwing knives.
  6. Created a few cutscenes and lore tapes.
  7. Reverted all UI elements back to a flat, 2D HUD
  8. Created the 5th Build of the game.
  9. Refined the in-game menus

AUGUST 2024

V.DEV-8.24

  1. Added basic ground troop soldier AI features such as patrolling and shooting
  2. Created a dynamic armor system where the player has a plate of armor per body region. If any damage is taken in that area after the plate breaks, damage will be applied to their actual health. Amount of damage is also applied according to the body segment damaged.
  3. Added the framework for 3 player classes. (Assassin, Commando, Cybersoldier). Framework as in modified attributes and stats.
  4. Added a TON of detail to the GDD, which has reached a peak of over 10,000 words. Actual story and lore are now beginning to take shape, as well as a much larger scope for the game.
  5. Refined the in game-menus

SEPTEMBER 2024

V.DEV-9.24

  1. Made this website (instead of working on the game ik)
  2. Added a fully functioning save/load system, along with the ability to create different save slots.
  3. Added 3 Cybersoldier Hacks. This is a special mechanic unique to the Cybersoldier class.
  4. Added 2 new types of enemy spacecraft, along with adding 4 types of Starship upgrades that consists of new mechanics and weapons, along with a solid progression system.
  5. Started work on a new map that will be designed specifically for the new scope of the game. It's located on the frozen polar ice caps of Mars, where a Soviet laboratory is suspected of doing experiments on frozen alien lifeforms. Most of the map will be designed using modular parts.
  6. Continued to polish the FPS gameplay
  7. Refined the in game-menus

OCTOBER 2024

V.DEV-10.24

  1. Added a cool warping sequence that teleports you to the target planet or moon.
  2. Added a 3D interactive landing menu
  3. Added realistic atmospheres to the appropriate planetary bodies
  4. Started work on the Basic Training level (The Tutorial).
  5. Reworked the way missions are handled.
  6. Replaced the stylized, computer interface user interface with a more futuristic, sci-fi UI.
  7. Added semi-smart NPC's with their own dialog, backstory, and ranking.
  8. Continued to work on this website a little bit

NOVEMBER 2024

V.DEV-11.24

  1. Created a playable, working prototype of a tutorial, with smart NPCs that teach the player the basics of my game.
  2. Replaced the mixamo skeletal mesh of the NPCs to an actual modern astronaut mesh.
  3. Replaced the old astronaut arms with way better ones
  4. Added the 357 Magnum Weapon (fully works, fully custom animated, etc.)
  5. Added a compass to the HUD
  6. Continued to update a lot of the hud elements
  7. Updated the overall look of this website, to better match the new UI style of the game.
  8. Created the 6th build of the game
  9. Added the setup screen, where the player customizes their personal AI, and sets some basic settings.
  10. Made the orbiting menu more realistic by adding a static sun, dynamic sunlight intensity/sizes, and a new Earth texture that better fits the story of the game.

DECEMBER 2024

V.DEV-12.24

  1. Tweaked some of the post processing effects a bit
  2. Added pedestrian NPCs that crowd different environments. These NPCs have randomized Metahuman faces.
  3. Added the framework for a leveling system to each separate weapon
  4. Started work on a abandoned Russian City map, with an exploration mission giving the player the freedom to explore to their liking.
  5. Added dynamic sounds to shooting and walking on different materials.
  6. Completely revamped the Jetpack system, replacing the overheating bar with multiple Jetpack uses. The jetpack in general is also way better, more fun, and viable to use.
  7. Added a new exoplanet, Proteus. It's a water world, and the first exoplanet discovered by humanity.
  8. Completely remade the Starship movement, it now feels more grounded, feeling more like a jet than a rocket.

JANUARY 2025

V.DEV-1.25

  1. Continued to work on the abandoned Russian City Map (adding new meshes, flags, extra details, etc)
  2. Developed the 7th build of the game, and ran the game's first proper play test, which taught me many things that needed to be prioritized, like proper optimization.
  3. Created the Basic AI States for the Watchdog Spider-bot enemy
  4. Created a foundation for easily creating multiple new ships.
  5. Added the CT-Night Eagle ship
  6. Remade the Starship HUD UI
  7. Started working on a new intro mission (I think the one that is mostly complete is too boring/bland)

FEBRUARY 2025

V.DEV-2.25

  1. Continued to work on the new intro level, by allowing the player to choose their starter class/profession, and name, in a cool interactive cutscene.
  2. Added a lore system to the game, the player can now found VHS Tapes, CDs, or files that they can view later
  3. Added a few quality of life things to make FPS gameplay feel more smooth
  4. Improved the design of a few UI elements
  5. Tweaked the post processing effects for the game
  6. Added a status UI, to effectively tell the player which effects are currently applied (Bleeding out, broken leg, etc.)

MARCH 2025

V.DEV-3.25

  1. Added an Extreme Radiation Hazard, that was built upon the status UI I added last month.
  2. Improved the Starmap and Dev menus a bit
  3. Added a physically accurate model of Earth to the Title Screen with proper time, and sun location synchronization.
  4. Added a proper Settings menu with Video, Graphics, Gameplay, Audio, Input, and UI options.
  5. Re-designed the saves menu
  6. Started working on a new mission, and added a minigame for it. This minigame is similar to one of those aerial bombing missions you'd see in Call of Duty.
  7. Began the process of replacing the orbit level with mission hubs. I plan on creating 6 of these levels/mission hubs, for all 6 of the (planned as of right now) playable worlds.
  8. Added procedurally generated foliage along with a custom landscape material for Polaris Island (Intro Mission)
  9. Made the Soviet Drop Pods rain down from space for Polaris Island (Intro Mission)

APRIL 2025

V.DEV-4.25

  1. Started work on a new map, Grand Nova. This is a strictly commercial zone, where the player will be able to buy many items from many different vendors.
